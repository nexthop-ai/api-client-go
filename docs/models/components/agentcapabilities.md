# AgentCapabilities

Describes which protocol features the agent supports. In addition to the standard capabilities (prefixed with ap.), implementations can declare custom capabilities, named in reverse domain notation (eg. com.example.some.capability).


## Fields

| Field                                                                                                                            | Type                                                                                                                             | Required                                                                                                                         | Description                                                                                                                      |
| -------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------- |
| `ApIoMessages`                                                                                                                   | **bool*                                                                                                                          | :heavy_minus_sign:                                                                                                               | Whether the agent supports Messages as input/output/state. If true, the agent uses the `messages` key in threads/runs endpoints. |
| `ApIoStreaming`                                                                                                                  | **bool*                                                                                                                          | :heavy_minus_sign:                                                                                                               | Whether the agent supports streaming output.                                                                                     |
