# TimePeriodType

Type of time period for which to run the report/policy. PAST_DAY is deprecated.


## Values

| Name                     | Value                    |
| ------------------------ | ------------------------ |
| `TimePeriodTypeAllTime`  | ALL_TIME                 |
| `TimePeriodTypePastYear` | PAST_YEAR                |
| `TimePeriodTypePastDay`  | PAST_DAY                 |
| `TimePeriodTypeCustom`   | CUSTOM                   |
