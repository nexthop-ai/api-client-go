# ToolSets

The types of tools that the agent is allowed to use. Only works with FAST and ADVANCED `agent` values


## Fields

| Field                                                                              | Type                                                                               | Required                                                                           | Description                                                                        |
| ---------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------- |
| `EnableWebSearch`                                                                  | **bool*                                                                            | :heavy_minus_sign:                                                                 | Whether the agent is allowed to use web search (default: true).                    |
| `EnableCompanyTools`                                                               | **bool*                                                                            | :heavy_minus_sign:                                                                 | Whether the agent is allowed to search internal company resources (default: true). |
