# DlpReportStatus

The status of the policy/report. Only ACTIVE status will be picked for scans.


## Values

| Name                       | Value                      |
| -------------------------- | -------------------------- |
| `DlpReportStatusActive`    | ACTIVE                     |
| `DlpReportStatusInactive`  | INACTIVE                   |
| `DlpReportStatusCancelled` | CANCELLED                  |
| `DlpReportStatusNone`      | NONE                       |
