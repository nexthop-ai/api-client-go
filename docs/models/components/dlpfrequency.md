# DlpFrequency

Interval between scans. DAILY is deprecated.


## Values

| Name                     | Value                    |
| ------------------------ | ------------------------ |
| `DlpFrequencyOnce`       | ONCE                     |
| `DlpFrequencyDaily`      | DAILY                    |
| `DlpFrequencyWeekly`     | WEEKLY                   |
| `DlpFrequencyContinuous` | CONTINUOUS               |
| `DlpFrequencyNone`       | NONE                     |
